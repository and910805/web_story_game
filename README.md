# 後宮·臭寶傳

一款基於中國宮廷背景的文字冒險遊戲（Visual Novel），融合戀愛模擬與劇情選擇元素。

## 🎮 遊戲特色

- 💕 **溫馨幽默的劇情**：宮鬥背景下的甜蜜愛情故事
- 🎭 **多重選擇系統**：玩家的每個選擇都會影響劇情走向和寵愛值
- 👑 **角色成長**：從秀常在到皇貴妃的晉升之路
- ✨ **三種結局**：根據玩家選擇達成不同的結局
- 🎨 **精美中國風UI**：金色漸層、宮廷裝飾、動畫效果
- 💾 **存檔功能**：隨時存檔，繼續遊戲

## 📖 遊戲簡介

**遊戲類型**：角色扮演 / 文字冒險 / 戀愛模擬  
**遊戲時長**：約 1 小時  
**語言**：繁體中文

> 「在這險惡的後宮中，我吳芃秀求的不是榮華富貴，而是那個叫臭咘咘的男人心裡的唯一位置。」

在雍正二十五年，臭咘咘皇帝登基，你作為秀女吳芃秀入宮。本來只想在家躺平的你，卻陰差陽錯地得到了皇帝的寵愛。在這個充滿挑戰的後宮中，你需要做出各種選擇，既要應對綠茶貴妃的陷害，又要維持與皇帝的關係……

## 🎯 遊戲章節

1. **序章：選秀風雲** - 初次見面，給皇帝留下印象
2. **第一章：初入宮闈與「椒房獨寵」** - 在御花園的意外相遇
3. **第二章：後宮風波 - 綠茶的陷阱** - 面對陷害，如何自保？
4. **第三章：冷宮（？）與真心** - 真情流露的考驗
5. **第四章：封后大典前夕** - 最終選擇，決定結局

## 🏆 結局類型

- **聖母皇太后** - 成為賢良淑德的皇后，與帝伉儷情深
- **私奔到月球** - 放棄皇位，與愛人環遊世界
- **歲月靜好，臭寶不老** - 完美結局，獲得真愛

## 🎮 操作說明

- 點擊「繼續 ▶」按鈕推進劇情
- 在關鍵時刻選擇對話選項
- 不同的選擇會影響：
  - 💕 寵愛值
  - 📊 角色屬性（美貌、智慧、體力）
  - 👑 封號晉升
  - 🎭 最終結局

## 🚀 部署到 GitHub Pages

### 方法一：直接上傳文件

1. 在 GitHub 創建一個新的 repository
2. 將以下文件上傳到 repository：
   - `index.html`
   - `style.css`
   - `game.js`
   - `README.md`
3. 在 repository 的 Settings → Pages 中：
   - Source 選擇 `main` 分支
   - 點擊 Save
4. 等待幾分鐘，訪問 `https://你的用戶名.github.io/repository名稱/`

### 方法二：使用 Git 命令

```bash
# 初始化 git repository
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 後宮·臭寶傳遊戲"

# 添加遠端 repository
git remote add origin https://github.com/你的用戶名/repository名稱.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

然後在 GitHub Settings → Pages 啟用 GitHub Pages。

- 🎭 **沉浸式體驗**：漸層背景、玻璃形態效果

## 🎯 遊戲系統

### 屬性系統
- **封號**：隨劇情晉升（秀常在 → 秀嬪 → 秀皇貴妃）
- **寵愛值**：影響結局的關鍵數值
- **美貌**：角色魅力值
- **智慧**：影響某些選擇的效果
- **體力**：角色精力值

### 存檔系統
- 使用瀏覽器 LocalStorage
- 可隨時存檔/讀檔
- 支持斷點續玩

## 📝 開發說明

此遊戲完全使用靜態網頁技術開發，無需後端服務器，可以直接在瀏覽器中運行。

### 本地測試

直接用瀏覽器打開 `index.html` 即可開始遊玩。

### 修改故事內容

在 `game.js` 文件中的 `gameData` 對象裡修改：
- `chapters`：各章節的場景和對話
- `endings`：結局內容
